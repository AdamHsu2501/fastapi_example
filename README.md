## FastAPI example

1. Clone repo
   ```
   git clone https://github.com/AdamHsu2501/fastapi_example.git
   ```

2. Install env
    ```
    pipenv install
    ```
3. start env
   ```
   pipenv shell
   ```
4. Run app
   ```
   <!-- uvicorn [model]:[var] --reload -->
   uvicorn main:app --reload
   ```
